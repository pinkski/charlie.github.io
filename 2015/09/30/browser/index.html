浏览器内核基础到进阶

Sep 30, 2015

学习浏览器内核需要会什么？

当我们写好了html、css、javascript发到服务器，浏览器从服务器获取这些资源，通过浏览器内核对html css javascript解析并渲染呈现

浏览器内核是什么？

成分

JS引擎越来越独立，内核倾向于单独指渲染引擎

渲染引擎

Trident(IE内核)
  • 1997年IE4首次采用
  • IE8的JS引擎是Jscript，IE9开始JS引擎是Chakra
  • win10后，IE浏览器命名为Edge，采用新内核EdgeHTML
Gecko(Firefox内核)
  • Netscape6首次使用
  • 开源
  • 跨平台，可在win、Linux、MacOS运行
Webkit(Safari)
  • 是KHTML的一个开源分支
  • 相比Gecko，源码结构清晰和渲染速度极快
  • JS引擎是JavaScriptCore（KJS的分支）
Blink(Chrome)
  • 2008谷歌发布chrome浏览器，内核命名chromium
  • 2013年谷歌宣布chromium与Webkit分离，由Webkit衍生出Blink
  • 由于和苹果竞争关系，Blink删除了880w行Webkit代码
  • JS引擎是V8

为什么用浏览器内核？

无论PC，还是移动端都离不开浏览器的存在

浏览器内核怎么用？

移动端内核

  • iPhone : Webkit
  • Android4.4之前 : Webkit
  • Android4.4之后 : Blink
  • Windows Phone8 : Trident
